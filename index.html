abcjs — local usage

Rendering ABC notation with your local clone at proj/abcjs

→ Open ABC Sheet Music UI

Quick start

From the proj directory, start a local server:

cd ~/proj
./start-abcjs.sh

Then open http://localhost:8080 in your browser.

Minimal HTML

<div id="paper"></div>
<script src="abcjs/dist/abcjs-basic.js"></script>
<script>
  const abc = `T: Cooley's
M: 4/4
L: 1/8
K: Emin
|:D2|EB{c}BA B2 EB|~B2 AB dBAG|FDAD BDAD|FDAD dAFD|
EBBA B2 EB|B2 AB defg|afe^c dBAF|DEFD E2:|`;

  ABCJS.renderAbc("paper", abc);
</script>

Core API

FunctionPurpose
ABCJS.renderAbc(target, abcString, params?)Render sheet music into a DOM element or selector. Returns an array of tune objects.
ABCJS.renderAbc(target, abcString, { responsive: "resize" })Re-layout when the window is resized.
ABCJS.renderAbc(target, abcString, { oneSvgPerLine: true })One SVG per staff line — best for printing / PDF export.
new ABCJS.Editor(textareaId, options)Live editor: type ABC, see notation update.
ABCJS.synth.CreateSynth()Browser audio playback (requires user gesture to start).
ABCJS.strTranspose(abc, steps)Transpose ABC text by semitone steps.
